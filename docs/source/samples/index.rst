Sample Applications
====================

The supplied Sample Applications can give a practical overview of the Embever IoT library.
The goal is to provide ideas about possible implementations, using them as a template project and clarify
the functionalities.

Each sample application is a standalone IoT application using the Arduino framework.
Because of that, porting them to another framework is straightforward.
To see more, select one of the sample applications.
